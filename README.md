# 🌶️ Spicy Store

A modern e-commerce web application built using React, designed to deliver a smooth shopping experience with fast performance, clean UI, and scalable architecture.

---

## 🔗 Live Demo

👉 https://shafikulnayem10.github.io/Spicy_Store/

---

## 🎯 Project Goal

The objective of this project was to build a fully functional frontend e-commerce application that focuses on:

- Smooth user experience and navigation  
- Efficient state management for cart operations  
- Scalable component-based architecture  
- Responsive and visually appealing UI  

This project demonstrates how real-world e-commerce platforms handle **product browsing, routing, and cart interactions**.

---

## ✨ Key Features

- 🛒 Add and remove products from the cart  
- 🔍 Real-time search functionality  
- 📄 Dynamic product detail pages  
- 🧭 Seamless navigation using React Router  
- 💾 Persistent cart data using localStorage  
- 🎨 Fully responsive and modern UI  

---

## 🧠 How It Works (Technical Breakdown)

- Built using **React functional components and hooks**  
- Implemented **client-side routing** with React Router DOM  
- Managed cart state dynamically with `useState`  
- Used **localStorage** to persist cart data across sessions  
- Created reusable UI components for scalability and maintainability  

---

## ⚙️ Tech Stack

| Technology | Why It Was Used |
|---|---|
| React.js | Component-based architecture for scalable UI |
| Vite | Fast build tool and optimized development workflow |
| Tailwind CSS + daisyUI | Rapid UI development with consistent design system |
| React Router DOM | Enables dynamic routing and multi-page experience |
| GitHub Pages | Simple and reliable deployment for frontend apps |
