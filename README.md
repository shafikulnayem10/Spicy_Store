# 🌶️ Spicy Store

**Spicy Store** is a modern e-commerce web application built using React, focused on clean UI, smooth navigation, and fast performance.

## 🚀 Tech Stack

- **React.js** – Component-based architecture  
- **Vite** – Fast bundler and development server  
- **Tailwind CSS + daisyUI** – Responsive and modern UI  
- **React Router DOM** – Multi-page routing  
- **GitHub Pages** – Deployment

## 🔗 Live Demo

👉 https://shafikulnayem10.github.io/Spicy_Store/

## ✨ Features

- 🛒 Add and remove items from the cart  
- 🔍 Search functionality for quick product access  
- 🧾 Dynamic product detail pages  
- 🧭 Smooth navigation using React Router  
- 🎨 Fully responsive design  

## 📈 Learning Outcomes

- Deploying React SPAs with routing on GitHub Pages  
- Managing local state and dynamic routes  
- Building scalable UIs using reusable components
